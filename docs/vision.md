# Proyecto: Smart Home Energy Simulator

## Objetivo

Desarrollar una aplicación web interactiva que permita a los usuarios estimar el consumo eléctrico de una vivienda mediante una representación visual.

A diferencia de los simuladores tradicionales basados en formularios, el usuario podrá navegar por los ambientes de la vivienda, agregar dispositivos y visualizar el impacto energético en tiempo real.

## Problema

La mayoría de simuladores de consumo eléctrico muestran únicamente tablas y formularios, dificultando que el usuario comprenda qué espacios y dispositivos generan mayor consumo.

## Solución

Representar una vivienda digital donde cada habitación tenga sus propios dispositivos eléctricos, indicadores de consumo y estadísticas.

## Público objetivo

* Usuarios residenciales.
* Familias.
* Estudiantes.
* Empresas eléctricas.
* Instituciones educativas.

## Objetivos del producto

* Calcular consumo diario.
* Calcular consumo mensual.
* Calcular costo estimado.
* Mostrar consumo por habitación.
* Educar sobre eficiencia energética.
* Identificar dispositivos de alto consumo.

## Colores que debes de utilizar 

1. Azul Principal (Brand / Primary)
Código hexadecimal: #1565C0
Uso: Color institucional (Electro Puno) utilizado para los elementos de marca, botones principales, enlaces activos y anillos de enfoque (ring).
2. Naranja de Acento (Accent)
Código hexadecimal: #F58635
Uso: Utilizado como color de acento para destacar información importante, bordes (como el del sidebar), textos pequeños de categorías (overline) y elementos interactivos secundarios.
3. Fondo Claro y Neutros (Light Background & Neutrals)
Código hexadecimal: #FFFFFF (Fondo) y tonos Slate/Zinc (ej. slate-900 para textos principales, slate-200 para bordes).
Uso: Establece el fondo limpio del modo claro y la jerarquía de los textos y separadores.
4. Fondo Oscuro (Dark Mode Background & Panels)
Códigos hexadecimales: #161C24 (Fondo base) y #2C3640 (Tarjetas y contenedores).
Uso: Define la paleta de colores para el tema oscuro, proporcionando un aspecto moderno de bajo contraste
